My Skils

CSS (Cascading Style Sheets) este un limbaj de stiluri folosit pentru a descrie prezentarea și formatarea HTML și a altor limbaje de marcare.

CSS este utilizat pentru a defini aspectele de aspect vizual ale unei pagini web, cum ar fi aspectul de layout, culorile, fonturile și multe altele

HTML (HyperText Markup Language) este limbajul de marcare standard utilizat pentru crearea paginilor web și a aplicațiilor web.

Acesta oferă un set de etichete și atribute care pot fi utilizate pentru a defini structura și conținutul unei pagini web, inclusiv text, imagini, link-uri, formulare, tabele și multe altele.

JavaScript este un limbaj de programare utilizat în principal pentru crearea de pagini web și aplicații web interactive.

Este un limbaj de programare client-side, ceea ce înseamnă că rulează pe browser-ul utilizatorului și nu pe server.
